2. 再读 `功能说明` 和 `使用手册`，快速上手业务操作。

## 当前功能完成度（本阶段）

- 已完成：
  - LLM 多方案配置管理（Alias 唯一 ID）
  - Model Hub 页面配置与保存
  - Model Switch 页面切换 active alias 与删除方案
  - TOML 文件双向同步（页面改文件、文件改页面）
  - 基础测试（后端 pytest + 前端构建）
- 未完成（后续优化项见改进计划）：
  - 前端自动化测试覆盖
  - 权限与审计
  - 安全加固（密钥加密存储）
  - 容器化部署链路联调
