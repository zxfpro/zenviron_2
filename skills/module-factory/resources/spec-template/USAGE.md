# Usage

## Quick Start

1. Install:
   - `npx skills add <repo> --skill module-<slug> -y`
2. Copy module payload:
   - `cp -R ~/.agents/skills/module-<slug>/resources/module/* <target-project-dir>/`
3. Apply integration micro-adjustments from `INTEGRATION.md`.
4. Run tests and E2E.
