# 测试用例与验收

## 1. 自动化测试
运行命令：
```bash
uv run pytest
```

已覆盖场景（核心）：
- 邮箱注册 -> 登录 -> `/auth/me`
- 忘记密码 -> 重置 -> 新旧密码验证
- 手机发码 -> 手机登录
- 验证码不可复用
- 未登录访问 `/auth/me` 返回 401
- 密码哈希与 JWT 编解码

## 2. 手工验收建议
1. Web 页面登录切换是否正常
2. 邮箱注册验证码流程完整
3. 忘记密码后旧密码不可登录
4. 手机验证码登录可获得 JWT
5. 退出后 token 清理

## 3. 通过标准
- 全量测试通过
- 覆盖率 >= 80%（当前已达标）
- 关键接口无 5xx
