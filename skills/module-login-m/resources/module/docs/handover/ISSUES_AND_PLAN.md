# 当前问题与改进计划

## 1. 当前问题（v1.0）
1. 手机短信未直连火山引擎官方接口（当前为 mock/webhook 桥接）。
2. 缺少统一审计日志（登录失败、风控命中、短信发送结果）。
3. 缺少管理员可视化运维页（仅容器和日志方式排查）。
4. 认证安全策略可增强（IP 黑名单、设备指纹、异常登录告警）。

## 2. v1.1 改进计划（建议两周内）
1. 接入火山短信官方 SDK/API，替换 webhook 桥接。
2. 增加短信发送状态落库（成功/失败/错误码）。
3. 增加登录安全策略：失败次数限制、滑动窗口限流。
4. 增加 API 观测指标（请求量、错误率、延迟）。

## 3. v1.2 改进计划
1. 支持刷新 Token / 会话管理。
2. 支持短信模板与签名配置后台化。
3. 补全 E2E 自动化（页面登录全链路）。
